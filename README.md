# Personal Notes App (NoteKeeper)

A full-stack note-taking application built with **React**, **Node.js**, **Express**, and **MongoDB**. Users can create, view, update, delete, and search notes efficiently.

---

## **Features**

- Create notes with multiple sections (heading + subheading + content)  
- Edit and delete notes  
- Search notes by keyword (heading, subheading, or content)  
- Responsive UI with Tailwind CSS  
- Persistent storage using MongoDB  

---

## **Technologies Used**

### Backend
- Node.js & Express.js  
- MongoDB & Mongoose  
- dotenv  
- CORS  

### Frontend
- React.js  
- React Router DOM  
- Context API for state management  
- Tailwind CSS for styling
